GUESS OR DRINK

Fun art, not fine art

This game is a fun twist on the classic game Pictionary. Please note drinking is not mandatory and players should drink responsibly if choosing to drink. The Game Rules:
* For a set number of round players will take turns drawing a given word

* Each player will have 90 seconds to draw the word.
* All other players are guessers and have 60 seconds (after 30 seconds has elapsed) to guess what is being drawn.
* Points gained by the guessers are equivalent to the time remaining when the word is guessed.

* For each person who guesses the word correctly the player drawing will also receive an amount of points relative to the number of players.

* If played using drunk mode and used as a drinking game, along side the normal rules:

- Given at least one player guesses the word any guessers who did not guess should drink.

- If no guessers guess the word the person drawing should drink.
